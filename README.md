## Movement
|Keys        |Result                |
|------------|----------------------|
|`W`         |moving up             |
|`A`         |moving left           |
|`S`         |moving down           |
|`D`         |moving right          |
|`esc`       |open/close inventory  |
|`ctrl + c`  |open/close console    |
