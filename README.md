## Movement
|Keys        |Result                |
|:----------:|----------------------|
|`W`         |moving up             |
|`A`         |moving left           |
|`S`         |moving down           |
|`D`         |moving right          |
|`ESC`       |open/close inventory  |
|`Ctrl-C`    |open/close console    |
